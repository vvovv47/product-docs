```

    ┌─────────────────────┐
    │    [耳机底部视图]    │
    │                     │

    │    ●      ●         │  ◄── 充电触点 (2pin)
    │                     │

- 触点1: VCC (充电正极)

- 触点直径: Φ1.5mm
- 触点间距: 3mm
```
