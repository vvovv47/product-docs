**唤醒响应机制** [推理]

```
唤醒响应流程:

用户: "小优小优"
         ↓
机器人: 
  1. LED灯语提示（蓝色闪烁）
  2. 头部转向声源方向
  3. 语音回应："我在，请说"
  4. 进入监听状态
         ↓
用户: "把那个红色的料箱搬到A区"
         ↓
机器人:
  1. 意图识别：搬运任务
  2. 槽位提取：物体=红色料箱，目标=A区
  3. 确认："好的，我将把红色料箱搬到A区"
  4. 执行任务
```
